Sistema operativo:
Es el software base que permite ejecutar Node.js y gestionar los recursos del computador o servidor.

Node.js:
Es un entorno que permite ejecutar JavaScript fuera del navegador y construir servidores web.

Aplicación server.js:
Es el archivo principal donde se define la lógica del servidor y las rutas de la API.

Puerto:
Es el canal por donde el servidor recibe solicitudes, en este caso el puerto 3000.

Logs:
Son registros que muestran la actividad del servidor, errores y eventos importantes.
