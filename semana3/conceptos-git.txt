Repositorio:
Un repositorio es un espacio donde se almacenan los archivos y cambios de un proyecto. Permite organizar el trabajo y mantener un historial de modificaciones realizadas durante el desarrollo.

Commit:
Un commit es un registro de los cambios realizados en un proyecto. Cada commit guarda una versión específica del trabajo y permite identificar qué modificaciones se hicieron.

Push:
El comando push permite enviar los cambios del repositorio local hacia GitHub u otro repositorio remoto. De esta manera, los archivos quedan publicados y respaldados en línea.
