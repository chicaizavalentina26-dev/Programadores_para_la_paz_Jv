Al enviar una petición POST a http://localhost:3000/registro con el JSON:

{
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

El servidor devuelve la siguiente respuesta:

{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

Esto demuestra que el servidor recibe correctamente los datos del cuerpo de la petición mediante req.body.
